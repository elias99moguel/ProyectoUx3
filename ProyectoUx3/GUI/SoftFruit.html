SoftFruit

La aplicación SoftFruit esta enfocado en la gestión de papayas en grandes cantidades, para fácilitar el uso de los pedidos, comunicación entre proveedor y cliente todo en la misma aplicación. Seguimos trabajando para traer un mejor servicio.

Programado por: Pedro Pablo Estrella


¿Para quién va dirigido?

La aplicación SoftFruit va dirigido al publico que trabaje con las papayas en grandes cantidades, ya que esta hecha para usar con código QR, tener contacto con los proveedores.

¿En que lenguaje esta programado?

La aplicación móvil esta desarrollado en Dart con el framework Flutter para poder ser una aplicación hibrida, con esto quiero decir que se pueda usar tanto en IOS como en Android.
